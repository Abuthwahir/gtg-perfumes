# GTG Perfumes

A responsive product landing page built with pure HTML, CSS, and JavaScript from a Figma design.

## Live Demo

[View Live Site](https://abuthwahir.github.io/gtg-perfumes/)

## Overview

This project was created as a front-end development assignment focused on converting a Figma layout into a responsive, interactive, and cleanly structured webpage without using frameworks.

The implementation follows a static front-end approach using only:

- HTML
- CSS
- JavaScript

## Features

- Responsive layout for desktop, tablet, and mobile
- Fixed header with in-page navigation
- Mobile hamburger menu
- Interactive product gallery with:
  - thumbnail selection
  - arrow navigation
  - dot navigation
  - smooth transitions
- Fragrance selection logic
- Expandable subscription cards
- Dynamic add-to-cart link and selection summary
- Add-to-cart confirmation message
- Newsletter subscription feedback message
- Search panel for quickly locating sections and fragrance options
- Animated percentage counters using `IntersectionObserver`
- Accordion section for collection details
- Comparison table built with semantic HTML table markup
- Image assets sourced from the provided design exports

## How To Run Locally

1. Clone or download the project.
2. Open the folder in VS Code or any editor.
3. Run it with a local static server such as Live Server.
4. Open `index.html` in the browser.

## Key Implementation Notes

- Built without React, Vue, Bootstrap, Tailwind, or any external framework
- Uses modular vanilla JavaScript for UI interactions
- Uses exported local assets for the main visual sections
- Includes smooth scrolling and section linking for navigation
- Includes cache-busted CSS and JS references for GitHub Pages refresh reliability

## Assignment Focus

This project was developed to demonstrate:

- pixel-accurate front-end implementation
- responsive design handling
- DOM interaction and state updates in vanilla JavaScript
- clean code organization
- practical UI behavior without a framework
